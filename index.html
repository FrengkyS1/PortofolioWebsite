02

About

I’m a computer-science student based in Indonesia. I tend to build for myself first—if a tool annoys me, I’d rather spend a weekend rewriting it than keep complaining about it.

Lately that means a lot of native desktop work: C++ when I want to be close to the metal, Rust and Tauri when I want it to ship without a 200 MB runtime. I’m comfortable on the web too—React, TypeScript, a bit of Python on the backend—but the things I’m proudest of run on the desktop.

I’m not done learning any of it. If you’ve read this far, you probably get the appeal of building the unglamorous thing properly.
