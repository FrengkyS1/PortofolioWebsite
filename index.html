I’m Frengky. I build small native desktop apps: the media tools I wished already existed, then keep using them myself.

I’m a computer-science student in Indonesia. Most of what I know I picked up by breaking things and reading other people’s source code. I care about apps that start fast, stay out of the way, and don’t ship a browser inside them.

Currently learning Rust properly and rewriting my old C++ player one module at a time.

Selected work

  1. 2025 Desktop · native

    LuminaPlayer

    A video player I wrote in C++ on top of libmpv, because I wanted something light that just plays my files without the bloat. It has its own library view, a thin rendering layer over the mpv engine, and keeps its settings in plain JSON. No Electron, no web view, no telemetry.

    • C++20
    • libmpv
    • CMake
    • Win32

  2. 2025 Desktop · Tauri

    LuminaDL

    A desktop video downloader built with Tauri. A small Rust core does the actual work while a React front-end stays deliberately quiet. It lives in the system tray, so it’s there when I need it and invisible when I don’t. This one taught me most of the Rust I know.

    • Tauri 2
    • Rust
    • React 19
    • TypeScript
    • Tailwind

    Source →

  3. 2025 Browser extension

    Video Extension

    A browser extension I wrote because the built-in volume cap wasn’t enough for quiet embeds, especially iframes like ok.ru. It boosts per-tab audio beyond 100 %, remembers the level across page loads, and adds keyboard shortcuts (Z / X) to rewind or skip forward 10 seconds without touching the mouse. Small popup, two tabs, nothing more.

    • JavaScript
    • HTML
    • Web Extensions API

    Source →

  4. 2026 Desktop · Tauri

    Hearth

    A control panel for setting up a Windows desktop the way I like it: installing the tools (a tiling window manager, status bar, patched fonts), writing their config files, and toggling services and startup items, all from one window instead of a dozen manual steps. A Rust backend does the privileged work while the front-end stays plain JavaScript, no framework. I built it because I was tired of redoing the same ritual on every fresh install.

    • Tauri 2
    • Rust
    • JavaScript
    • CSS

About

I’m a computer-science student based in Indonesia. I tend to build for myself first: if a tool annoys me, I’d rather spend a weekend rewriting it than keep complaining about it.

Lately that means a lot of native desktop work. C++ when I want to be close to the metal, Rust and Tauri when I want it to ship without a 200 MB runtime. I’m comfortable on the web too (React, TypeScript, a bit of Python on the backend), but the things I’m proudest of run on the desktop.

I’m not done learning any of it. If you’ve read this far, you probably get the appeal of building the unglamorous thing properly.
